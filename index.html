Model Weights,
Losslessly Compressed

A running project on making model weights smaller without changing what they compute. Most model compression trades bits for quality; lossless avoids that trade entirely: nothing rounded, nothing pruned, the same model to the last digit. Measured on NVIDIA’s Nemotron-3 family. Current progress:
